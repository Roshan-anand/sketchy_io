# sketchy
sketchy.io is a free online multiplayer drawing and guessing pictionary game. A normal game consists of a few rounds, where every round a player has to draw their chosen word and others have to guess it to gain points!. The person with the most points at the end of the game, will then be crowned as the winner!
Have fun!
